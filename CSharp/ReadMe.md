# C# Cheatsheet

Welcome to the C# cheatsheet! This guide will help you quickly get up to speed with C# programming.

## Overview

C# is a modern, object-oriented programming language developed by Microsoft. It's widely used for building Windows applications, web applications with ASP.NET, games with Unity, and more.

- [Intermediate.md](Intermediate.md) - OOP principles, collections, LINQ, and error handling
- [Advanced.md](Advanced.md) - Async/await, delegates, events, and advanced patterns

## Quick Reference

### Basic Syntax
```csharp
// Single line comment
/* Multi-line comment */

// Namespace and class declaration
namespace MyApp
{
    class Program
    {
        static void Main(string[] args)
        {
            Console.WriteLine("Hello, World!");
        }
    }
}
```

### Key Features

- **Strongly Typed**: Type safety at compile time
- **Object-Oriented**: Full OOP support with classes, interfaces, and inheritance
- **Cross-Platform**: Run on Windows, Linux, and macOS with .NET 6+ (Long-Term Support)
- **Garbage Collection**: Automatic memory management
- **Rich Standard Library**: Extensive built-in functionality

## Getting Started

1. Install [.NET SDK](https://dotnet.microsoft.com/download)
2. Create a new project: `dotnet new console -n MyApp`
3. Run your application: `dotnet run`
